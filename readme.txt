== Description ==

Provides a shortcode to embed an Ordnance Survey map with waypoints.

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the `wp-os-walks` folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.
